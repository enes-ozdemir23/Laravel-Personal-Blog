# 📝 Laravel — Admin Panelli Kişisel Blog Sistemi

**Laravel** framework kullanılarak geliştirilmiş, 
tam kapsamlı admin paneline sahip dinamik blog uygulamasıdır.

---

## 🚀 Kullanılan Teknolojiler

| Teknoloji | Açıklama |
|-----------|----------|
| PHP | Ana programlama dili |
| Laravel | Web framework |
| Blade | Laravel şablon motoru |
| MySQL | Veritabanı |
| HTML / CSS / SCSS | Arayüz |
| JavaScript | Dinamik işlemler |
| Bootstrap | Responsive tasarım |

---

## ⚙️ Özellikler

### 🖥️ Admin Paneli
- ✅ Blog yazısı ekleme, düzenleme, silme
- ✅ Yazıları aktif / pasif yapma
- ✅ Sayfa ve alan ekleme, düzenleme, silme
- ✅ Resim yükleme ve yönetimi
- ✅ Site logosu yükleme ve değiştirme
- ✅ İçerik yönetim paneli

### 🌐 Kullanıcı Arayüzü
- ✅ Blog yazılarını listeleme ve okuma
- ✅ Dinamik sayfa yapısı
- ✅ Responsive tasarım
